## Recommended capture settings

- **Format:** PNG.
- **Dimensions:** capture at a desktop width of ~1440px (roughly 1440×900), 2×
  device-pixel-ratio for a crisp result on high-DPI displays. Crop to the relevant
  UI so the image isn't dominated by empty space.
- Use realistic but non-sensitive content, and avoid showing real account
  emails/usernames or any secrets.
